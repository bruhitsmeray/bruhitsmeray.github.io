Introduction

Welcome

Aleph is a learning project originally based on 4.26, it is packed with mostly C++ code and BPs to help you kickstart your game project.

Aleph and this documentation represents the project based on Unreal Engine 4.27.2

NOTE: A future upgrade to Unreal Engine 5 will come at a later date!

Game Instance

In Aleph a lot of important stuff is kept inside the game instance. Some variables might be moved in another files later down the line. Here you can also find some functions to help you access Subtitles, Notifications and Tasks with more ease. In the game instance you can also find a drop-down menu that lets you select the HUD that will be used in-game.
