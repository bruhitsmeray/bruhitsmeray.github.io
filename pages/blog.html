CATCHING UP (again)

So, it's been nearly 1 year since the last "catching up" and then I kinda ghosted this blog, so yeah... sorry about that. I'd like to talk about the blog design first. Some might say that the previous design looked better and it might be true, but now it's simpler. Anyway, this year I've worked at a few scripts and projects, let's get the scripts out of the way first.

automatr:

automatr & cleanr are 2 powershell scripts for cleaning Windows and installing some essential (at least for me) apps. That's it, done. You can find it on GitHub.

Eutera 4.5:

Eutera 4.5 is an updated version of the Discord bot. Functionally it's not too different but it uses Discord.JS 13 and approaches some tasks differently from the previous versions. I might bring it back online at a later date.

Aleph:

Now let's get to the new stuff, Aleph is a project that I've been contributing to for the past few months, and it serves as a great basis for Decay and other future games. Aleph serves more as a compilation of code and blueprints. Helping any related development going forward. So that's pretty nice.

Decay:

Well this is an old-new project, started in 2019. Now it's using most of Aleph's code, if not improved a little to fit the needs of the game. Decay is still in early development, with a demo coming at a later date.

I think this is it for now, just wanted to keep you posted with progress on stuff.
